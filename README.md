## English

MeetFlow is a professional, open-source screen recording software designed for educators and professionals. It features an intelligent hardware probing algorithm to automatically detect and utilize the best GPU encoder (NVIDIA, AMD, Intel), ensuring high-performance recording without requiring technical expertise.

### Key Features
* 🧠 **Smart Hardware Acceleration**: Auto-detects the best GPU encoder on startup.
* 🛡️ **Crash-Proof Recording**: Uses `.mkv` containerization to prevent file corruption during unexpected crashes or power losses.
* 🎯 **Commercial-Grade UI**: Built with CustomTkinter for a modern, responsive user experience.
* 🔄 **Built-in Remuxer**: Lossless conversion from MKV to MP4.

### Code Signing
Releases of this project are digitally signed to ensure authenticity and prevent anti-virus false positives. 
**This project uses the [SignPath Foundation](https://signpath.org/) for code signing.** We thank them for supporting the open-source community!

### Privacy Policy
**MeetFlow runs strictly locally on your machine.** 
We do not collect, store, transmit, or process any user data, telemetry, or recording files. Your data remains 100% yours.

## 繁體中文

MeetFlow 是一款專為教育工作者與專業人士設計的開源商業級螢幕錄影軟體。它內建智慧硬體探測演算法，能在啟動時自動偵測並套用最佳的 GPU 編碼器 (NVIDIA, AMD, Intel)，讓使用者無需具備專業技術背景，也能享受極致順暢的錄影效能。

### 核心功能
* 🧠 **智慧硬體加速**: 軟體啟動時自動鎖定最佳 GPU 編碼器，大幅降低 CPU 負載。
* 🛡️ **防崩潰錄影架構**: 強制採用 `.mkv` 容器格式，確保在遭遇系統當機或斷電時，已錄製的影片檔案絕對不會損毀。
* 🎯 **商用級介面**: 使用 CustomTkinter 打造具備現代感與高互動性的操作介面。
* 🔄 **內建轉檔器**: 錄製完畢後提供無損轉檔，一鍵將 MKV 轉換為主流的 MP4 格式。

### 數位簽章與安全性
為了確保軟體的真實性並防止防毒軟體誤判，本專案的發布版本皆具備數位憑證簽章。
**本專案感謝 [SignPath Foundation](https://signpath.org/) 提供開源社群程式碼簽章服務！**
