# MeetFlow Screen Recorder 🎥

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Build](https://github.com/actions/workflows/build_and_sign.yml/badge.svg)

MeetFlow is a professional, open-source screen recording software designed for educators and professionals. It features an intelligent hardware probing algorithm to automatically detect and utilize the best GPU encoder (NVIDIA, AMD, Intel), ensuring high-performance recording without requiring technical expertise.

## Key Features
* 🧠 **Smart Hardware Acceleration**: Auto-detects the best GPU encoder on startup.
* 🛡️ **Crash-Proof Recording**: Uses `.mkv` containerization to prevent file corruption during unexpected crashes or power losses.
* 🎯 **Commercial-Grade UI**: Built with CustomTkinter for a modern, responsive user experience.
* 🔄 **Built-in Remuxer**: Lossless conversion from MKV to MP4.
